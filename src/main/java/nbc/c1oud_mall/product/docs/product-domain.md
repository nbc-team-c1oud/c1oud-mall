# Product Domain Docs

## API 예외 정책
- 단건 조회 시 존재하지 않는 상품을 요청하면 `ErrorCode.PRODUCT_NOT_FOUND(PRD001)`을 통해 404 응답을 반환합니다.


## 상품 데이터 초기화 정책 (Dummy Data)
- **적용 범위:** 로컬 및 개발(`dev`) 환경에서만 동작하며, 운영(`prod`) 환경에서는 절대로 수행되지 않음 (`@Profile("dev")` 강제).
- **자동 적재:** 애플리케이션 컨텍스트 로드 직후 `CommandLineRunner`를 통해 기본 샘플 상품 3건이 자동 삽입됨.
- **멱등성(Idempotency) 보장:** 서버 재시작 시 이미 데이터가 존재하면(`count() > 0`) 중복 삽입을 방지하는 방어 로직이 적용되어 있음.
- **주의사항:** 운영 DB 스키마 생성 및 데이터 적재는 이 초기화 빈을 사용하지 않고, 반드시 별도의 Migration 툴(Flyway 등)이나 관리자 도구를 통해 수행한다.
